Automated plant watering and voting system

This Project was inspired by the discontinued Take Care Of My Plant project. Each day a reddit bot which I coded using Javascript and the reddit API creates a post on which users vote whether or not they want the plant to be watered. This bot analyzes each users comment (only if it is on the correct post) and decides if this is a yes vote a no vote or an invalid input. this bot is run on a raspberry pi. At the end of each voting cycle the bot tallies up the votes and if the majority is yes, it then turns on an LED on an Arduino UNO micro-controller which is hooked up to a relay. Plugged into this relay is an air pump which pumps water out of a jug and onto the plant. This project currently has around 250 participants.

Pulse Social Media

Pulse Social Media is a platform I created on which users can share the daily occurrences of their lives either anonymously or through a registered account. users can also use the integrated search bar to look up other user's profiles if they like the content which they are posting. This was created using HTML, PHP and MYSQL databases. Pulse is more as a desktop site, but can also be viewed on an phone.
